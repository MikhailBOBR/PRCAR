# About Project

## Purpose

`PRCAR` was developed as a web application for automating the work of an online car dealership. The system includes both the customer-facing part of the website and the internal business logic for employees.

Customers can browse the car catalog, open detail pages, save cars to favorites, and submit requests. Managers can add and edit cars and process customer requests. Administrators can manage user roles.

## Implemented Features
- public car catalog
- car detail pages
- registration and authentication
- customer account
- favorites
- purchase, booking, and test-drive requests
- request status management
- catalog management
- user role administration
- image storage
- automated testing

## Result

The result is a fullstack project that models a real online car dealership with role-based access control, a database, and server-side business logic.
