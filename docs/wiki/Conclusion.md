# Conclusion

During the development of `PRCAR`, a fullstack web application for an online car dealership was created. The project includes a public catalog, a customer account, favorites, order requests, a manager panel, and an admin module.

The system was built with `Next.js`, `TypeScript`, `Prisma`, and `PostgreSQL`. It also includes image storage support, role-based access control, and automated tests for the key business logic.

The final result demonstrates practical use of modern web development tools, database interaction, client-server architecture, and structured business logic.
